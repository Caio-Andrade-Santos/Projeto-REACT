Projeto React — Gerenciamento de Frotas
 Descrição

Projeto desenvolvido em grupo (4 integrantes) utilizando React, com foco no gerenciamento de frotas de veículos.
A aplicação permite cadastrar veículos com validações e armazenar os dados por meio de uma API, onde uma das rotas utiliza um array para guardar os veículos adicionados.

 Funcionalidades

Cadastro de veículos

Validações dos dados de entrada

Integração com API para gerenciamento da frota

 Tecnologias Utilizadas

React

JavaScript

API REST

Git e GitHub

 Objetivo do Projeto

Aplicar conceitos de desenvolvimento Front-End, consumo de APIs, trabalho em equipe e organização de código.
